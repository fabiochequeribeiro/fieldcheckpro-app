# Política de Privacidade - FieldCheck Pro

Última atualização: 19 de junho de 2026

O FieldCheck Pro é um aplicativo para gestão de checklists, visitas técnicas, ocorrências e levantamentos de peças em campo.

## Dados tratados
O aplicativo pode tratar dados de login, identificação de técnicos, clientes, obras, equipamentos, registros de checklist, ocorrências, fotos, assinaturas, relatórios e arquivos exportados.

## Finalidade
Os dados são usados para registrar atividades técnicas, gerar relatórios, apoiar o acompanhamento de serviços e sincronizar informações entre o aplicativo e o ambiente de dados configurado pelo responsável pela conta.

## Compartilhamento
Relatórios e arquivos só são compartilhados quando o usuário utiliza funções de envio, exportação, e-mail ou compartilhamento do próprio dispositivo.

## Armazenamento
Parte dos dados pode ser armazenada localmente no dispositivo para permitir uso em campo e evitar perda de informações. Quando configurado, os dados também podem ser sincronizados com Supabase.

## Permissões
O aplicativo pode solicitar acesso à câmera e imagens para anexar fotos aos registros técnicos. O acesso à internet é usado para login, sincronização e envio de dados.
